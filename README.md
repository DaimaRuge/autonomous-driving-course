# 自动驾驶与机器人算法 - 完整课程计划

> 📅 制定时间：2026-03-14
> 👨‍🏫 制定人：Joy (王子乔)
> ⏱️ 总时长：78 周（~18 个月）
> 🎯 目标：培养具备自动驾驶与机器人算法全栈能力的工程师

---

## 📋 课程概览

### 核心能力
1. **编程基础**：C++、Python、Linux、Git、Docker
2. **算法基础**：数据结构、算法、数学、CI/CD
3. **ROS/ROS2**：机器人操作系统开发
4. **感知算法**：传统 CV + 深度学习 + 3D 感知
5. **控制算法**：经典控制 + 现代控制
6. **路径规划**：全局 + 局部 + 行为决策
7. **大模型应用**：LLM + VLM + World Models
8. **仿真实践**：CARLA + Gazebo
9. **应用实战**：无人驾驶 + 服务机器人 + 工业机器人

### 学习节奏
- **每日学习**：2-3 小时
- **每周学习**：10-18 小时（5-6 天）
- **每月任务**：1-2 个模块 + 1-2 个项目

## 🗓️ 课程时间表（78 周）

### 阶段 1：基础夯实（Week 1-8，2 个月）
- C++ 基础语法
- C++ STL 与模板
- Python 基础
- Linux + Git + Docker

### 阶段 2：算法基础（Week 9-16，2 个月）
- 数据结构
- 基础算法
- 数学基础
- Git + CI/CD

### 阶段 3：ROS/ROS2 开发（Week 17-20，1 个月）
- ROS 基础
- ROS2 进阶

### 阶段 4：感知算法（Week 21-32，3 个月）
- 传统计算机视觉
- 深度学习基础
- 3D 感知
- BEV 与 Transformer
- 多传感器融合
- SLAM

### 阶段 5：控制算法（Week 33-40，2 个月）
- 经典控制算法
- 现代控制算法
- 强化学习控制
- 端到端控制

### 阶段 6：路径规划算法（Week 41-48，2 个月）
- 全局路径规划
- 局部轨迹规划
- 动态避障
- 行为决策规划

### 阶段 7：大模型应用（Week 49-56，2 个月）
- LLM 基础
- VLM（视觉-语言模型）
- World Models
- 大模型在自动驾驶中的应用

### 阶段 8：仿真实践（Week 57-68，3 个月）
- CARLA 仿真
- Gazebo 仿真
- 算法验证与优化

### 阶段 9：应用实战（Week 69-78，2.5 个月）
- 无人驾驶应用
- 服务机器人应用
- 工业机器人应用

## 📖 推荐资源

### 书籍
- **基础**：《C++ Primer》、《Python 编程：从入门到实践》、《算法导论》
- **感知**：《Computer Vision: Algorithms and Applications》、《Deep Learning for Vision Systems》
- **控制**：《Feedback Control of Dynamic Systems》、《Model Predictive Control》
- **规划**：《Planning Algorithms》、《Principles of Robot Motion》
- **大模型**：《Deep Learning》、《Natural Language Processing with Transformers》

### 在线课程
- **基础**：CS61A, CS124, CS168
- **感知**：CS231N, CS223A, CS234
- **控制**：EC 500, ME 541, EECS C149
- **规划**：CS 224, CS 287
- **大模型**：CS224N, CS25

### 开源项目
- **仿真**：Apollo Auto, Autoware, CARLA, Gazebo
- **深度学习**：PyTorch3D, mmdetection3d, BEVFusion
- **机器人**：ROS 2, Navigation 2, MoveIt
- **大模型**：LLaMA, CLIP, BLIP, Stable Diffusion
